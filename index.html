Step-by-Step Attack Narrative
01
Initial Compromise
Attacker exploits unpatched Apache Struts 2 RCE (CVE-2013-2251 / CVE-2018-11776) on internet-exposed application servers. Remote code execution achieved without credentials. Affected: DUO01, HSD46, JUM02, SCA01.
02
Privilege Escalation
Attacker leverages Windows Print Spooler (PrintNightmare, CVE-2021-34527) and Windows Unquoted Service Path (CVE-2014-0759) to escalate from standard user to SYSTEM-level privileges on domain-joined servers.
03
Lateral Movement
Using harvested credentials and NTLM relay attacks — enabled by SMB Signing not enforced across 40+ servers and all 54 endpoints — attacker moves laterally across the entire server estate.
04
Domain Compromise
Attacker reaches Domain Controller, extracts Active Directory credentials, and achieves full domain administrator access. Backup server (BKP01) accessible without authentication — primary ransomware target.
05
Data Exfiltration
Financial records, ERP credentials, SAP data, employee PII, and operational databases exfiltrated to attacker infrastructure. Encrypted TLS 1.0 channels (SWEET32) allow exfiltration without detection.
06
Business Disruption
Ransomware deployed across enterprise. ERP systems, file servers, and backup infrastructure encrypted. GCC operations halted. Shared services delivery to global business units disrupted. HSD72 (Sophos gap) — no EDR alert fires.
